1) What is method overloading?
   Method overloading means having multiple methods with the same name
   but different parameters (type or count) in the same class.

2) How does Java pass parameters?
   Java always uses pass-by-value. Copies of variables are passed to methods.

3) Difference between return and void?
   return sends a value back to the caller.
   void does not return any value.

4) What is stack memory?
   Stack memory stores method calls, local variables, and references.
   It follows LIFO (Last In First Out).

5) Why is modular code important?
   It improves readability, reusability, debugging, and maintenance.
